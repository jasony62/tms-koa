# 安装依赖包

> cnpm i neo4j-driver

> https://www.npmjs.com/package/neo4j-driver

> https://neo4j.com/docs/api/javascript-driver/current/

# 配置文件

| 属性     | 类型    | 说明               |
| -------- | ------- | ------------------ |
| disabled | boolean | 关闭配置           |
| master   | object  | neo4j 服务连接参数 |
| -        |         |                    |
| host     | string  | neo4j 服务主机地址 |
| port     | number  | neo4j 服务主机端口 |
